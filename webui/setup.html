Open GPU Monitor any time from the ● near the top-right of your menu bar.
        A few things it does:

      
    
        
  • ✏️ Rename people — right-click any username in the popover and
          type their real name. It replaces that username everywhere, for good.
    • 
        
  • Copy usernames — the ⧉ button next to a name copies the raw username.
    • 
        
  • Availability alarms — get a banner + sound when a GPU frees up (any idle GPU,
          ≥ X GB free, or specific types like 1×A100 + 2×H100). Open the 🔔 in the popover.
    • 
        
  • Menu-bar colour — the dot is 🟢 when a whole 80GB GPU is free, 🔴 when nothing is,
          🟠 in between. The number is how many GPUs are idle.
    • 
        
  • 15-minute sparklines — recent utilisation on every GPU card.
